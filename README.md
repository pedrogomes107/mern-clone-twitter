# Clone Twitter - Backend

Tutorial: https://www.youtube.com/watch?v=4GUVz2psWUg

## Install npm packages:

#### `npm install express mongoose jsonwebtoken bcryptjs dotenv cors cookie-parser cloudinary`

#### `npm install -D nodemon`

## Run Server Locally:

#### `npm run dev`

## Deploy App:

#### `npm run build`

#### `npm run start`
